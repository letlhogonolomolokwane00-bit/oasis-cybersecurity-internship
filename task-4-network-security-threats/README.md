**What I learned**



This task helped me understand that network attacks do not all have the same purpose. A DDoS attack mainly affects the availability of a service, while a Man-in-the-Middle attack targets communication between two parties. Spoofing is used to make a message, device or service appear trustworthy.



I also learned that organisations need several security controls working together. These include encryption, traffic monitoring, firewalls, multi-factor authentication, regular updates and employee awareness.
