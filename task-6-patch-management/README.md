**What I learned**



This task helped me understand that patch management involves more than installing updates. An organisation must know which systems it owns, identify available patches, assess the risk, test important updates and confirm that installation was successful.



I also learned that sending a patch notification is not enough. The WannaCry and Equifax incidents showed why organisations need clear responsibility, accurate records and proper verification.
